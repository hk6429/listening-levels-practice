# 傾聽五層次 · 練習場（即時同步版）

七個習慣｜習慣五「知彼解己」課堂互動工具。學生透過**房間密碼 + 五位數座號**進入同一房間，全班即時共用煩惱池，隨機抽籤練習五個傾聽層次。

## 功能
- 🔐 房間密碼 + 五位數座號（年級1+班級2+座號2，如 `10512`）進入
- 📱 平板/手機/筆電通用，多人即時同步
- ✍️ 文字或語音輸入煩惱，全班可見（顯示座號 + 內容）
- 🎲 隨機抽一則煩惱 + 一個傾聽層次（0–4）
- 🎤 語音練習回應（Web Speech API，Chrome/Edge 佳）
- 📥 匯出煩惱池成 .txt
- 🗑 一鍵清空整個房間

## 首次安裝
見 [FIREBASE_SETUP.md](./FIREBASE_SETUP.md)（約 5 分鐘）

## 使用流程
1. 老師開網址 → 房間密碼 + 座號 → 進入
2. 學生平板開相同網址 → 同一房間密碼 + 自己座號
3. 輸入煩惱 → 全班看到 → 按抽籤
4. 三人一組：說話者／傾聽者／觀察者輪替
5. 課後老師清空房間

## 傾聽五層次
| 層次 | 名稱 | 本質 |
|------|------|------|
| 0 | 聽而不聞 | 不聽 |
| 1 | 虛應故事 | 不專注的聽 |
| 2 | 選擇性的聽 | 不專注的聽 |
| 3 | 專注的聽 | 站在自己的立場 |
| 4 | 同理心傾聽 | 站在別人的立場 |

## 技術
- 單一 HTML 檔
- Firebase Realtime Database（免費額度完全夠用）
- 無後端需要自架、無帳號登入
